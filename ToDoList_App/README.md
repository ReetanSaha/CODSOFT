# 📝 To-Do List Application

## 📌 Project Description

The To-Do List Application is a simple command-line Python project that helps users organize and manage their daily tasks. Users can add new tasks, view existing tasks, mark tasks as completed, and delete tasks. All tasks are stored in a text file, so they remain available even after the program is closed.

This project is designed for beginners to practice basic Python programming concepts such as functions, loops, conditional statements, lists, and file handling.

---

## 🚀 Features

- Add new tasks
- View all tasks
- Mark tasks as completed
- Delete tasks
- Save tasks automatically using a text file
- Beginner-friendly command-line interface

## ▶️ How to Run the Project

### Step 1: Open Terminal

Go to your project folder.

```bash
cd ~/Documents/ToDoListApp
```

### Step 2: Run the program

```bash
python3 todo.py
```
